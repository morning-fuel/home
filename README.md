## 📌 About the Project

**Morning Fuel** is a modern landing page for a fruit bowl delivery business.  
It promotes a healthy lifestyle with fresh, ready-to-eat fruit bowls.

The website is designed to:
- Showcase menu plans (Basic, Mid, Premium)
- Encourage daily healthy habits
- Convert visitors into customers via WhatsApp/Instagram

---

## ⚙️ Tech Stack

- React + Vite  
- Tailwind CSS  
- Framer Motion  
- Lucide Icons  
- GitHub Pages  

---

## 📦 Features

- Clean and modern UI  
- Fully responsive design  
- Scroll animations
